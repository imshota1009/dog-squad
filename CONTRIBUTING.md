# Contributing to DOG SQUAD: SQUIRREL SIEGE

Thank you for your interest in contributing! 🐾

## How to Contribute

### Reporting Bugs
1. Check the [Issues](https://github.com/imshota1009/dog-squad/issues) page to see if the bug has already been reported.
2. If not, open a new issue using the **Bug Report** template.
3. Provide as much detail as possible — browser, OS, steps to reproduce, and screenshots.

### Suggesting Features
1. Open a new issue using the **Feature Request** template.
2. Describe the feature clearly and explain why it would improve the game.

### Submitting Code Changes
1. Fork the repository.
2. Create a new branch: `git checkout -b feature/your-feature-name`
3. Make your changes and test them in the browser.
4. Commit with a clear message: `git commit -m "Add: description of change"`
5. Push to your fork: `git push origin feature/your-feature-name`
6. Open a Pull Request against the `main` branch.

## Code Style
- Keep JavaScript readable and commented where logic is non-obvious.
- CSS art characters and icons should follow the existing naming conventions in `css/style.css`.
- Test on both desktop and mobile browsers before submitting.

## Questions?
Feel free to open an issue with the label `question` if you need help understanding the codebase.
